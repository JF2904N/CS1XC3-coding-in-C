ls: list all surroundings 
cd (location): takes you to location
pwd: tells you where you are 
cd ..: returns to previous location 
cd ~: takes you back to main 
less: interacts with item
add (name): adds new item 
help: provides list of helpful commands
man: tells you how to use certain command 
mv: moves item to different location 
rm: removes item 
mkdir:creates new location
sudo:temporarily gives you more privelge 
grep (phrase) (item): search for the phrase in an item 
touch (item): creates new files 
cp (item)(newitem): makes a copy of an item
sudo less: provides more access to a files content
exit: terminate game
